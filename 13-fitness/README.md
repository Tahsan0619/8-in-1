# 13 — Fitness & Gym Platform

> A comprehensive fitness and gym management website with class schedules, trainer profiles, workout planner, BMI calculator, and membership plans — inspired by **ClassPass + Nike Training Club + MyFitnessPal** built as a standalone gym website for portfolio showcase.

---

## Design Philosophy

1. **Energetic & Bold** — High-contrast colours, strong typography, dynamic imagery
2. **Goal-Oriented** — Everything drives toward fitness goals and progress tracking
3. **Schedule-Centric** — Easy class discovery and booking
4. **Data-Driven** — BMI calculator, progress charts, workout logs with visual feedback
5. **Motivational** — Progress indicators, streak counters, achievement badges

---

## Tech Stack

| Layer | Choice |
|-------|--------|
| **Markup** | HTML5, semantic elements |
| **Styling** | CSS3 — Custom Properties, Grid, Flexbox |
| **Scripting** | Vanilla JS (ES6+) |
| **Charts** | Chart.js (progress, BMI, body stats) |
| **State** | localStorage for workouts, goals, bookings |
| **Icons** | Lucide Icons |
| **Fonts** | Google Fonts |
| **No frameworks** | Zero React / Vue / Tailwind — pure skill demo |

## What Makes This Different

- **Interactive class schedule** with weekly calendar and real-time booking
- **Workout planner** — create custom routines with exercise library, sets, reps, rest timers
- **BMI & body composition calculator** with visual chart output and health category
- **Progress tracking** with Chart.js graphs for weight, measurements, workout frequency
- **Membership comparison** with feature matrix and signup flow
- **Trainer profiles** with certifications, specialties, and class schedules
- **Achievement badges** and streak counters for motivation
- **BDT pricing** with Bangladeshi gym context
