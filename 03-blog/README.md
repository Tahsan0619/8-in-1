# Blog / Content Demo

A rich content platform inspired by the editorial craft of **The Verge**, the reading experience of **Medium**, the visual density of **Smashing Magazine**, and the personality of **Brain Pickings** — built for writers, readers, and curators.

## Tech Approach
- HTML5 / CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (ES6+ modules)
- Intersection Observer for reading progress + infinite scroll
- Web Audio API for podcast player
- No frameworks

## Design Philosophy
- **Typography is the UI** — reading experience above everything
- **Content density done right** — magazine layout without clutter
- **Reading-first** — distraction-free article view
- **Layered discovery** — trending, featured, categories all accessible without overwhelm
- **Audio-visual integration** — embedded video + podcast player as first-class

## Key Differentiators
- Reading progress bar on articles
- Sticky table of contents sidebar
- Drop cap typography on articles
- Threaded comment replies
- Embedded audio player (sticky bottom)
- Magazine editorial layout mode
- Breaking news ticker
- Exit-intent newsletter popup
- Clean reading mode toggle
