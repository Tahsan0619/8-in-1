# Portfolio Demo

A multi-mode portfolio experience inspired by the cinematic scroll of **Apple product pages**, the bold art direction of **Awwwards winners**, the precision of **Rauno Freiberg's** personal site, and the creative confidence of **Pentagram** — designed for developers, designers, photographers, and agencies.

## Tech Approach
- HTML5 / CSS3 (custom properties, grid, flexbox, scroll-snap)
- Vanilla JavaScript (ES6+ modules)
- Intersection Observer + requestAnimationFrame for smooth scroll effects
- CSS custom cursor
- No frameworks

## Design Philosophy
- **Personality-first** — the portfolio IS the project
- **Cinematic scroll** — the whole page is a curated experience
- **Bold typography** — display fonts, oversized headings
- **Pixel-perfect craft** — every detail matters (cursor, transitions, spacing)
- **Multi-mode** — personal, agency, photography all in one codebase

## Pages
| Page | Route |
|---|---|
| Home / Landing | `/` |
| About | `/about` |
| Work / Projects | `/work` |
| Project Detail | `/work/:slug` |
| Photography Gallery | `/photography` |
| Photography Album | `/photography/:album` |
| Testimonials | `/testimonials` |
| Contact | `/contact` |

## Key Differentiators
- Custom cursor animation
- Page transition animations (fade + slide)
- Scroll-hijack showcase section
- Before/after slider for redesign projects
- Masonry photo grid + lightbox
- Availability status badge
- Animated tagline (typewriter or morph)
- Full-screen project previews on hover
