# 15 — Nonprofit & Charity Platform

> A compelling nonprofit website with donation flow, project impact tracker, volunteer signup, campaign management, and fundraising progress bars — inspired by **GoFundMe + Charity:Water + GlobalGiving** built as a charity/NGO platform for portfolio showcase.

---

## Design Philosophy

1. **Emotional & Compelling** — Story-driven design that inspires action
2. **Transparency First** — Every donation tracked, every impact shown
3. **Low Friction Giving** — Donate in 3 clicks or less
4. **Community Building** — Volunteers, donors, and beneficiaries connected
5. **Trust & Credibility** — Verified status, financial transparency, impact reports

---

## Tech Stack

| Layer | Choice |
|-------|--------|
| **Markup** | HTML5, semantic elements |
| **Styling** | CSS3 — Custom Properties, Grid, Flexbox |
| **Scripting** | Vanilla JS (ES6+) |
| **Charts** | Chart.js (impact stats, donation trends, fund allocation) |
| **State** | localStorage for donations, campaigns, volunteers |
| **Icons** | Lucide Icons |
| **Fonts** | Google Fonts |
| **No frameworks** | Zero React / Vue / Tailwind — pure skill demo |

## What Makes This Different

- **Full donation flow** with amount presets, custom amount, dedication messages, and payment simulation
- **Real-time fundraising progress bars** with animated fill and donor count
- **Impact tracker** with Chart.js visualizations: funds raised, allocation pie chart, beneficiary stats
- **Campaign updates** timeline showing project milestones
- **Volunteer management** with opportunity matching and hour tracking
- **Fundraising events** with registration and countdown timers
- **Beneficiary stories** with before/after narrative format
- **Financial transparency** dashboard showing exactly where money goes
- **BDT currency** with Bangladeshi NGO context
