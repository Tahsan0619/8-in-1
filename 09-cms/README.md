# 09 — CMS / Website Builder

## Overview

A powerful **CMS and visual website builder** combining the intuitive drag-and-drop of **Webflow**, the content management of **WordPress**, and the design polish of **Framer** — featuring a live page builder, media library, blog/page management, SEO tools, and multi-site support. This is the ultimate portfolio showcase piece.

---

## Design Philosophy

- **WYSIWYG-first** — what you build is what you get, in real time
- **Builder as product** — the editing experience IS the product
- **Content-forward** — structured content types with clean editing UIs
- **Developer-friendly feel** — code-like precision in a visual interface
- **Dashboard clarity** — site stats, content status, and publishing at a glance

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 semantic elements |
| Styling | CSS3 — Custom Properties, Grid, Flexbox |
| Logic | Vanilla JavaScript (ES6+ modules) |
| State | LocalStorage for pages, posts, media, settings |
| DnD | Custom drag-and-drop engine (native Drag API) |
| Preview | Live iframe preview rendering |
| Icons | Lucide Icons (SVG) |

---

## Pages

| # | Page | Description |
|---|------|-------------|
| 1 | Dashboard | Site overview — stats, recent edits, quick actions, publish status |
| 2 | Page Builder | Visual drag-and-drop page editor with live preview |
| 3 | Pages Manager | List/grid of all pages — create, edit, duplicate, delete |
| 4 | Blog Posts | Post management — CRUD, scheduling, categories, tags |
| 5 | Post Editor | Rich text editor with media embeds, SEO meta, publish controls |
| 6 | Media Library | Upload, organize, search images/files — grid + list views |
| 7 | SEO Settings | Per-page meta title/description, OG tags, sitemap, robots config |
| 8 | Site Settings | General settings — site name, favicon, domain, analytics, custom code |
| 9 | Templates | Pre-built section/page templates to drag into builder |
| 10 | Theme Editor | Typography, colors, spacing — live preview theme customizer |

---

## What Makes This Different

- **Real Drag-and-Drop Builder:** Sections snap into place with visual guides
- **Live Preview Pane:** See changes instantly in a simulated browser frame
- **Content + Design Unified:** Not just a CMS — it's WordPress + Webflow in one
- **Template Library:** Pre-designed sections users can drop in and customize
- **SEO Dashboard:** Per-page SEO scoring with actionable recommendations
- **Media Manager:** Full Unsplash-style grid with tags, search, and drag-to-insert
- **No Framework:** Entire page builder in vanilla JS — ultimate skill demonstration
